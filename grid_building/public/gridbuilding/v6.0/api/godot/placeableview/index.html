UI view for a single placeable used in selection lists. Renders a placeable’s icon and display name in a horizontal layout with configurable sizing and padding. Emits PlaceableSelected when the user clicks on the view. Usage: var view = PlaceableView.New(); view.Placeable = myPlaceable; view.FixedViewHeight = 48; view.FixedIconSize = 40; view.PlaceableSelected += _OnPlaceableSelected;

Fixed icon size for consistent icon dimensions across all placeable views. When set to a positive value, enforces both width and height of icon TextureRect. When set to 0, icon sizing is not enforced and will use scene file settings. Default: 40 pixels to match standard icon sizing.
