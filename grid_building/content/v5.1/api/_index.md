# C# API Reference

This section contains the complete C# API reference for the Grid Building Plugin v5.1.

## Namespaces

The API is organized into the following namespaces:

- **GridBuilding.Core.Base** - Core enums and interfaces
- **GridBuilding.Core.Contexts** - State management contexts  
- **GridBuilding.Core.Resources** - Resource classes and settings
- **GridBuilding.Core.Utils** - Utility classes
- **GridBuilding.Shared.Enums** - Shared enumerations

Browse the namespaces on the left to find detailed documentation for each class, interface, and enum.
