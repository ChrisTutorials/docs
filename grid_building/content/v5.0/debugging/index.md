Use these tools to visualize and troubleshoot runtime state.

## Targeting

- [GridTargetingDebugText](../api/GridTargetingDebugText/) — shows current cell, world position, and snap mode
- [TargetHighlighter](../api/TargetHighlighter/) — highlights the active cell or region

## Validation

- [GBLogger](../api/GBLogger/) — route validation diagnostics to console/log
- [PlacementReport](../api/PlacementReport/) — inspect aggregated rule outcomes

## Indicators

- [IndicatorSetupReport](../api/IndicatorSetupReport/) — summarize indicator configuration and status
